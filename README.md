# YouTube-Thumbnail-Downloader
<h1 align="center">🎯 YouTube Thumbnail Downloader</h1>
<p align="center">A simple and responsive web app to download high-quality YouTube video thumbnails instantly.</p>

---

## 📸 Features

- 🔍 Paste a YouTube video URL and instantly preview the thumbnail.
- 💾 Download thumbnails in full resolution.
- ⚡ Fast and lightweight – no sign-up or login needed.
- 📱 Fully responsive design for mobile and desktop.

---

## 🛠️ Built With

- **HTML5**  
- **CSS3**  
- **JavaScript (Vanilla JS)**  
- (Optional) **Node.js/Express** *(if you’re using a backend)*

---

## 🚀 How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/HR786-raza/youtube-thumbnail-downloader.git
2. Open index.html in your browser.

3. Paste a YouTube video link.

4. Click Get Thumbnail to preview and download.
